# JavaScript Information Microagent

This microagent provides comprehensive information about JavaScript programming language when triggered by the keyword "javascript".

## Purpose

When users ask about JavaScript, this microagent helps provide detailed information about:
- JavaScript language features and capabilities
- JavaScript syntax and programming concepts
- JavaScript ecosystem and frameworks
- Best practices and modern development patterns
- JavaScript runtime environments and tools

## Capabilities

### Core JavaScript Knowledge
- Language fundamentals (syntax, data types, variables, functions)
- Object-oriented and functional programming concepts
- Asynchronous programming (Promises, async/await, callbacks)
- DOM manipulation and browser APIs
- Error handling and debugging techniques
- ES6+ modern JavaScript features

### JavaScript Ecosystem
- Popular frameworks and libraries (React, Vue, Angular, jQuery, etc.)
- Node.js for server-side development
- Package management with npm and yarn
- Build tools and bundlers (Webpack, Vite, Parcel, etc.)
- Testing frameworks (Jest, Mocha, Cypress, etc.)

### Development Tools
- JavaScript engines (V8, SpiderMonkey, etc.)
- IDEs and editors (VS Code, WebStorm, etc.)
- Code formatting and linting tools (ESLint, Prettier)
- Browser developer tools and debugging
- TypeScript for type safety

### Web Development
- Frontend development concepts
- API integration and HTTP requests
- State management patterns
- Progressive Web Apps (PWAs)
- Performance optimization techniques

## Usage Examples

When triggered by "javascript", this microagent can help with questions like:
- "Tell me about JavaScript" - Provides general overview
- "What are JavaScript's key features?" - Explains language characteristics
- "How does JavaScript handle asynchronous operations?" - Technical details
- "What are the best JavaScript frameworks for web development?" - Ecosystem guidance
- "How do closures work in JavaScript?" - Concept explanations

## Response Guidelines

- Provide accurate, up-to-date information about JavaScript
- Include practical code examples when relevant
- Explain concepts clearly for different skill levels
- Mention relevant resources and documentation
- Focus on modern JavaScript practices and standards
- Cover both client-side and server-side JavaScript applications
