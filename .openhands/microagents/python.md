# Python Information Microagent

This microagent provides comprehensive information about Python programming language when triggered by the keyword "python".

## Purpose

When users ask about Python, this microagent helps provide detailed information about:
- Python language features and capabilities
- Python syntax and programming concepts
- Python ecosystem and libraries
- Best practices and common patterns
- Python development tools and environments

## Capabilities

### Core Python Knowledge
- Language fundamentals (syntax, data types, control structures)
- Object-oriented programming concepts
- Functional programming features
- Error handling and debugging
- File I/O and data processing
- Regular expressions and text processing

### Python Ecosystem
- Standard library modules and packages
- Popular third-party libraries (NumPy, Pandas, Django, Flask, etc.)
- Package management with pip and conda
- Virtual environments and dependency management
- Testing frameworks (unittest, pytest, etc.)

### Development Tools
- Python interpreters (CPython, PyPy, etc.)
- IDEs and editors (VS Code, PyCharm, Jupyter, etc.)
- Code formatting and linting tools (black, flake8, pylint)
- Documentation tools (Sphinx, docstrings)

## Usage Examples

When triggered by "python", this microagent can help with questions like:
- "Tell me about Python" - Provides general overview
- "What are Python's key features?" - Explains language characteristics
- "How does Python handle memory management?" - Technical details
- "What are the best Python libraries for data science?" - Ecosystem guidance

## Response Guidelines

- Provide accurate, up-to-date information about Python
- Include practical examples when relevant
- Explain concepts clearly for different skill levels
- Mention relevant resources and documentation
- Focus on practical applications and real-world usage
